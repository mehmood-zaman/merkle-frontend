## Setup

1.  **Install dependencies.**

    Navigate into project directory and install all dependencies.

    ```shell
    cd Merkle-frontend
    npm install or yarn install
    ```

2.  **Open project.**

    All set? Your project is now running at http://localhost:3000

3.  **Technologies used in project.**

** React.js
** Redux.js
** Ant Design
** Styled-components
\*\* Axios

## Deployment

### Second option:
